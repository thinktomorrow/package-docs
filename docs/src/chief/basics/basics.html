# Introduction

In order to allow an administrator to manage the application, the content should be made controllable in the chief admin. This can be done by setting up a manager for your models. The implementation boils down to the following basics:

  • Preparing your model to be manageable
  • Creating a manager for this model
  • Registering the manager in the application
